# 🕵️ DeepDetect — Детектор дипфейк-изображений

> **DeepDetect** — микросервисное приложение для анализа изображений на наличие признаков deepfake. Загрузите фото через веб-интерфейс или Telegram-бота, и нейросеть оценит его подлинность.

## ✨ Возможности

- 🔍 **Анализ изображений** на наличие признаков deepfake с оценкой уверенности (0–100%)
- 🌐 **Веб-интерфейс** на React с адаптивным дизайном и поддержкой тёмной темы
- 🤖 **Telegram-бот** с поддержкой:
  - Отправки фото как изображения или файла
  - Кнопки для запуска **Telegram Mini App**
- 🐳 **Полная контейнеризация** — запуск одной командой `docker compose up`

## 🚀 Быстрый старт

### 📋 Предварительные требования

- [Docker](https://docs.docker.com/get-docker/) + [Docker Compose](https://docs.docker.com/compose/install/)
- Для Telegram-бота: имеющийся бот с токеном от [@BotFather](https://t.me/BotFather)
- Для Mini App: публичный HTTPS-URL (см. раздел [🌐 Настройка Mini App](#-настройка-mini-app))

### 1. Настройте переменные окружения

Создайте файл `.env` в корне проекта:

```bash
BOT_TOKEN=<токен_вашего_telegram_бота>

MINI_APP_URL=https://ваш.публичный.ip
```

### 2. Запустите систему

```bash
# Запуск всех сервисов в фоновом режиме
# нужно находиться в той же папке, что и docker-compose.yml (в корне проекта)
docker compose up --build -d

# Просмотр логов в реальном времени
docker compose logs -f

# Остановка системы
docker compose down

# Полная очистка (включая тома с временными данными)
docker compose down -v
```

## 🤖 Использование

### 🌐 Веб-интерфейс

1. Откройте в браузере: **http://localhost:8080**
2. Нажмите «Выберите изображение» или перетащите файл
3. Нажмите **«Анализировать»**
4. Дождитесь результата:
   - ✅ *Изображение подлинное* / ⚠️ *Обнаружен deepfake*
   - Индикатор уверенности в процентах

### 🤖 Telegram-бот

1. Найдите вашего бота в Telegram (по юзернейму из @BotFather)
2. Отправьте команду `/start`
3. Выберите способ:
   - 📤 **Отправьте фото** напрямую в чат (как изображение или файл)
   - 🌐 **Нажмите кнопку** «Открыть Mini App» для полноценного интерфейса (для этого в `.env` должен быть указан публичный адрес, на котором размещён сайт)
